# Reitinhakualgoritmit (Pyhton)

Testataan eri reitinhakualgoritmien tehokkuuksia

Tarkoituksena on rakentaa:
- Dijkstaran algorimi
- IDA* algoritmi

Algoritmeja testataan eri **verkoissa** ja mitataan **nopeutta**
